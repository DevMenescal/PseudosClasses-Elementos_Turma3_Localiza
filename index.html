Conhecendo Pseudo Elementos e Pseudo Classes

Pseudo Elementos

Os pseudo-elementos são usados para estilizar partes específicas de um elemento. Eles criam elementos virtuais que não existem no HTML, permitindo aplicar estilos a partes como a primeira letra de um texto, ou a primeira linha de um parágrafo. Alguns exemplos incluem:

::before

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Minima excepturi praesentium veritatis in iure distinctio magnam corporis consectetur cumque, qui corrupti recusandae dolore alias, consequuntur mollitia. Eaque fugiat at quam.

::after

Permite inserir conteúdo após o conteúdo do elemento selecionado.

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Error ipsa esse quam iure sed nam nulla repudiandae impedit! Fugiat cupiditate iure iusto expedita modi provident maxime sequi fuga, quasi ad!

::first-letter

Estiliza a primeira letra de um elemento de bloco.

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Vel, corporis placeat! Vitae ipsa, nisi ut rem nihil labore! Pariatur a neque nam consectetur, vitae quia dolore rem possimus adipisci doloremque!

::first-line

Estiliza a primeira linha de um elemento.

Lorem ipsum dolor sit, amet consectetur adipisicing elit. Ipsa dolores, dolore ut eveniet saepe qui suscipit esse commodi impedit? Vero atque eos consectetur. Porro, tenetur id alias voluptatibus laudantium eveniet!

Pseudo Classes

As pseudo-classes são utilizadas para aplicar estilos a elementos com base em estados específicos ou ações do usuário. Alguns exemplos comuns incluem:

:hover

:active

Aplica estilos quando o elemento está sendo cliclado. Ex:

:focus

Aplica um estilo quando o elemento tem foco, como após ser clicado ou selecionado por teclado.
